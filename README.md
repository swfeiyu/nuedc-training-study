+ 4月25日：晚上十点前在QQ群内发出自己github的ID
+ 4月25日——4月28日：软件内容的12和3开发环境搭建
+ 4月28日：晚上十点前在github仓库中使用md文件提交后两周的学习计划
+ 4月29日——5月12日：按照自己的计划学习硬件内容1234和软件内容3
+ 5月中旬：实验室电赛设计方案会议
+ 5月13日——5月19日：学习软件内容4并进行题目二方案制定

+ 6月3日——6月23日：进行题目一方案制定、开发和测试
